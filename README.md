# Password-Vault

A simple password vault website built with plain HTML, CSS, and JavaScript.

## Features
Add, edit, and delete saved passwords
- Search passwords by website or username
- Show/hide password visibility
- Copy password to clipboard

- Password strength meter

- All data saved in the browser using LocalStorage (stays after refresh)


## Files
1.index.html
2.style.css
3.script.js



## How to Run
 
1. Download all three files into the same folder.
2. Open `index.html` in any web browser.
3. Start adding passwords — no setup or installation needed.
