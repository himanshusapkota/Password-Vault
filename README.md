# 🔐 Password Vault

A simple, clean, client-side password manager built for the browser. Store, search, and generate passwords with a built-in strength checker — all in a distraction-free, dark-mode-friendly interface.

** Live Demo:** [himanshusapkota.github.io/Password-Vault](https://himanshusapkota.github.io/Password-Vault/)

---

##  Features

- **Add New Passwords** — Save credentials with a Website/App name, Username/Email, and Password.
- **Show/Hide Password** — Toggle password visibility while entering credentials.
- **Password Generator** — Instantly generate a strong, random password with one click.
- **Password Strength Meter** — Get real-time feedback on how strong your password is as you type.
- **Search Passwords** — Quickly find saved credentials without scrolling through the whole list.
- **Saved Passwords List** — View all stored entries in one organized place.
- **Dark Mode** — Switch between light and dark themes for comfortable viewing.

---

## 🛠️ Tech Stack

- **HTML5** — Page structure and markup
- **CSS3** — Styling, layout, and dark mode theming
- **JavaScript (Vanilla)** — App logic, password generation, strength calculation, and search/filtering
- **GitHub Pages** — Static site hosting

> No backend/server — the app runs entirely in the browser.

##  Getting Started

### Option 1: Use the Live Site
Just visit the [live demo](https://himanshusapkota.github.io/Password-Vault/) — no installation required.

### Option 2: Run Locally
1. Clone the repository
   ```bash
   git clone https://github.com/himanshusapkota/Password-Vault.git
   ```
2. Navigate into the project folder
   ```bash
   cd Password-Vault
   ```
3. Open `index.html` in your browser
   ```bash
   open index.html
   ```
   (or simply double-click the file / drag it into your browser)

No build steps or dependencies to install — it's a static site.

---

## 📖 Usage

1. Enter the **Website/App**, **Username/Email**, and **Password** in the *Add New Password* form.
2. Click **Generate** if you'd like a strong password created for you.
3. Check the **Strength** indicator to make sure your password is secure.
4. Click **Add Password** to save the entry.
5. Use the **Search Passwords** bar to quickly find a saved entry.
6. Toggle **Dark Mode** anytime for a different look and feel.
